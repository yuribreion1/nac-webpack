NAC2 - Optimization Technology

Abaixo, nesta página, você encontra um sistema de cálculo de notas de alunos. Esse sistema foi feito puramente com JavaScript ES5 (tradicional). Sua tarefa é replicar esse sistema para que utilize o ES6. Requisitos:

  • Configurar ES6 via webpack e npm
  • Divir o código em, pelo menos, dois arquivos e utilizar a notação ES6 para exportação e importação de códigos
  • Utilizar, pelo menos, 1 Arrow Function
  • Utilizar, pelo menos, 1 classe (class)
  • Utilizar a função map (de vetores)

Outras instruções você encontrará em: http://notapad.pw/optnac2
